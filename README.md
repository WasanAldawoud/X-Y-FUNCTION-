Function Definition
The function ( F(X, Y) ) is defined as follows:

If ( X >= 10 ) and ( Y >= 8 ): [ F(X, Y) = Y^3 + Y^3 + 300 ]

If ( X >= 3 ) and ( Y>= 7 ): [ F(X, Y) = Y^2 + Y^2 + X*Y ]

Otherwise: [ F(X, Y) = X + Y ]
